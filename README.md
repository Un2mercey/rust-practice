# rust-practice

## Calculator
```
cd calculator
```
### Run via cargo
```
cargo run
```
### Build release
```
cargo build --release
```

### Run release
```
target\release\calculator 1 + 2
```


## Image combiner

### Run combiner

```
cargo run -- images/1.jpg images/2.jpg images/output.jpg
```

then see result at the path `images/output.jpg`
